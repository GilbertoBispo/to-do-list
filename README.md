# To-Do List Full Stack
Ele foi desenvolvido com o objetivo de colocar em prática conceitos fundamentais de desenvolvimento web, unindo o front-end e o back-end em uma aplicação funcional. A ideia principal foi aprender "mão na massa", enfrentando os desafios de conectar uma interface de usuário a um servidor real.

## 🎯 O que eu aprendi:
-   **Front-end:** Como estruturar páginas com HTML5, estilizar com CSS3 e manipular o DOM de forma dinâmica e simplificada usando **jQuery**.
-   **Back-end:** Os primeiros passos com **Node.js** e o framework **Express**, entendendo como criar rotas e gerenciar requisições.
-   **Integração:** Como fazer o front-end "conversar" com o back-end, enviando e recebendo dados.
  
## 💻 Tecnologias Utilizadas

### Front-end
-   HTML
-   CSS
-   JavaScript (jQuery)
### Back-end
-   Node.js
-   Express
### Banco de dados
-   PostgreSQL
## 🚀 Como Executar
Siga os passos abaixo para rodar o projeto localmente:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com](https://github.com/GilbertoBispo/to-do-list)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o servidor:**
    ```bash
    npm start
    ```
